# Holon base package

Close Hermes, then run `install.ps1 -ConfirmHermesClosed`. Plugin activation is
separate and requires `-EnableHermesPlugin`. Uninstall preserves Holon data by
default; permanent data deletion additionally requires both `-RemoveData` and
`-ConfirmDataDeletion`.

M2.05 is a package foundation, not the final Wallet or release archive.
